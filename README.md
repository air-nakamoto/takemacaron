# タケマカロン (Takemacaron) 🍬

記録よりも「予算設計」を重視した、シンプルで可愛い家計簿アプリ。
「あといくら使えるか」を一目で把握し、無理のない範囲でお金を使う習慣をサポートします。

## 🎯 コンセプト
- **予算ファースト**: 月の総予算から「固定費」を引いた「実質使える金額（やりくり予算）」を明確にします。
- **視覚的なフィードバック**: 残りの予算に応じてマカロンのキャラクター（マスコット）の表情や色が変化します。
- **モバイル最適化**: スマホでの片手操作、素早い入力を最優先に設計されています。

## ✨ 実装済み機能 (Phase 1 + α)

### 1. 予算と固定費の管理
- **予算設定**: 月の総予算と、集計期間（開始日～終了日）を設定可能。
- **固定費管理**: 家賃やサブスクなどの固定費を登録・編集・削除できます。
- **自動計算**: `総予算 - 固定費合計 - 変動費（今の支出） = 残り予算` を自動で算出・表示します。

### 2. 支出の記録
- **クイック入力**: スマホ画面下部の「＋」ボタン（FAB）から、親指だけで素早く入力モードへ。
- **入力補助**: 日付、カテゴリ（選択式）、金額（数値キーボード）、メモに対応。
- **履歴リスト**: 直近の支出をリスト表示。編集・削除も可能です。

### 3. ビジュアル・フィードバック
- **残量ゲージ**: 予算を「バッテリー残量」のように表示。減っていくにつれて緑→黄→赤と変化します。
- **マスコット**: 予算の余裕度に合わせて表情が変わります（通常・焦り・パニックなど）。
- **マイルドな警告**: 予算オーバー時は、きつい赤色ではなく落ち着いたグレーで「OVER」と表示し、ストレスを軽減します。

### 4. 履歴・アーカイブ
- **自動アーカイブ**: 設定期間が過ぎると、自動的にその期間の記録を保存し、次の期間へ移行します。
- **履歴の確認**: 過去の予算達成状況（達成/超過）を一覧で確認、修正できます。

## 📱 操作性のこだわり
- **スマホ対応**: ヘッダーレイアウトの最適化、入力ボタンの配置、タップ領域の確保。
- **誤操作防止**: 削除時の確認ダイアログ、イベント伝播の制御。
- **PWAライク**: 入力時のズーム防止、独自スクロールバーなど、アプリらしい挙動。

├── App.tsx             # メインロジック（状態管理、計算、ルーティング的役割）

    ├── BudgetSettingsModal.tsx # 予算設定モーダル
    └── Mascot.tsx              # SVGマスコットコンポーネント
